## 0. 한 줄 방향
> **새 동네를 모험처럼 발견하되, 결국 "내가 살 하루"로 느끼게 한다.**
> 탐험은 호기심을, 일상은 친근감을 만든다. (탐험감=관광 / 일상감=정착)

핵심 전환: 게임의 골격을 **"명소 보물찾기"** → **"아이의 하루를 미리 살아보는 하루지도"** 로 재설계.
보물지도 수집 메커니즘(탐험감)은 유지하되, 결과물과 미션의 의미를 **일상 루틴**으로 바꾼다.

## 1. 요구사항 요약 (Requirements)
1. 맵은 명소 나열이 아니라 **아이의 하루 동선**(집→학교→방과후→생활→산책→집, 주말 바다)을 담는다.
2. **하루 루트 기반 반자유 탐험**: 순서가 있는 하루 흐름 + 각 단계의 선택지(도서관/놀이터 등).
3. **탐험감 장치**: 안개(fog-of-war)가 걷히며 장소 발견 + 보물(=일상카드) 조각 수집.
4. 미션은 영웅 서사가 아니라 **작은 하루 미션**(학교 가는 길 익히기, 방과후 아지트 찾기, 저녁 장보기 등).
5. 장소마다 **일상 카드** 지급 → 결과는 **"나의 속초 하루지도"** 내러티브로 출력.
6. 실제 사진은 **관광컷이 아닌 일상컷**(입구·횡단보도·벤치·진열대·교문·눈높이 길).
7. NPC는 **동네 사람들**(사서·시장 아주머니·버스기사·약사·산책 할아버지) + 상징 1종(갈매기).
8. 실제 속초 지리(행정동) 반영. 숨은 탐험지(유적·아바이마을·영랑호·영금정)는 주말/보너스.
9. (도구) 맵 구성 품질을 위해 **Tiled 기반 데이터 주도 맵**으로 전환(아래 §8).

## 2. 맵 설계 — 일상 구역 (실제 속초 지리 위에)
`DESIGN.md`의 지리(서=설악산, 동=동해, 중앙=청초호, 북=시장, 남=조양동 신시가지)를 **생활 구역**으로 재라벨링:

| 구역 | 위치(실제) | 장소 | 일상 의미 |
|------|-----------|------|-----------|
| 우리 집 주변 | 조양동 신시가지 | 집·골목·**횡단보도·버스정류장** | 매일 나가는 길 |
| 학교 구역 | 조양동/교동 | 학교·운동장·보건실·교문 | 매일 가는 곳 |
| 방과후 구역 | 조양동 | 도서관·놀이터 | 학교 끝나고 |
| 생활 구역 | 시내/조양동 | 시장(닭강정)·마트·분식집 | 가족 장보기·간식 |
| 산책 구역 | 청초호/영랑호 | 청초호 산책길 | 저녁·주말 산책 |
| 특별 탐험(주말/숨은) | 동해/조양동/청호동 | 해수욕장·유적·아바이마을·영금정 | 주말·발견 보너스 |

→ 월드맵 1장에 위 구역을 **권역으로 배치**, 핵심 동선(집-학교-방과후-생활-산책)은 길로 연결, 특별 탐험지는 외곽/안개 속.

## 3. 하루 진행 구조 (Day State Machine)
한 번의 플레이 = **속초의 하루**. 진행에 따라 안개가 걷히고 다음 구역이 열린다(반자유).

```
[아침] 집 → (학교 가는 길 미션) → [학교]
                                   ↓
[오후] 방과후 선택: 도서관 | 놀이터
                                   ↓
[저녁] 생활 선택: 시장 | 마트  → (저녁 산책: 청초호)
                                   ↓
[밤] 집으로 → 하루지도 정리
                                   ↓
[주말 해금] 바다(해수욕장) · 숨은 탐험지(유적/아바이/영랑호)
```

- 각 단계 클리어 시 **다음 구역의 안개 해제 + 일상 카드 획득**.
- 선택지는 "둘 중 하나"라도 둘 다 방문 가능(반자유). 진행은 막지 않되 **추천 동선**을 화살표/안내로 제시.

상태 모델(게임 state): `dayPhase` = `morning|school|afterschool|evening|night|weekend`, `cards[]`, `revealed{zoneId}`.

## 4. 일상 카드 & 결과 "하루지도"
보물 조각 → **일상 카드**로 의미 재정의(메커니즘 동일, 카피만 변경).

| 장소 | 카드 | 의미 |
|------|------|------|
| 집 | 아침 준비 카드 | 하루 시작 |
| 학교 가는 길 | 길 기억 카드 | 통학로 익힘 |
| 학교 | 새 학교 카드 | 매일 가는 곳 |
| 도서관 | 조용한 아지트 카드 | 쉬는 곳 |
| 놀이터 | 첫 친구 카드 | 친구 관계 |
| 시장 | 간식·장보기 카드 | 가족 생활 |
| 청초호 | 마음 쉬는 길 카드 | 안정 |
| 해수욕장(주말) | 주말 기대 카드 | 기대감 |

**결과 화면("나의 속초 하루지도")** — 카드들을 문장형 내러티브로:
```
아침에는 학교에 가요.
학교가 끝나면 도서관에 들를 수 있어요.
저녁에는 아빠와 시장에서 간식을 살 수 있어요.
주말에는 청초호를 걷거나 바다를 볼 수 있어요.
속초가 조금 내 동네처럼 느껴졌나요?
```
(획득한 카드에 따라 문장 동적 조합)

## 6. 사진 — 일상컷 기준
> "멋진 곳인가?"가 아니라 **"내가 거기서 어떻게 움직이는가?"** 를 보여주는 컷.
- 학교: 교문·통학길·운동장(특정 학교 식별 회피 → **샘플/일러스트**)
- 시장: 입구·간식가게 앞·골목·가격표·앉아 먹는 곳
- 도서관: 입구·어린이실·책장·안내데스크·신발장
- 청초호: 산책길·벤치·물·새
- 해수욕장: 해변 입구·산책로·모래 가까운 컷(항공샷 X)
- 개인정보/초상권: 사람·아이 식별 사진 회피, 일러스트/생성/공개자료 일상컷 우선.

각 장소 `lookMission`을 일상 관찰 항목으로 정비(예: 횡단보도·멈추는 곳·아빠 만날 장소).

## 7. 현재 코드 매핑 (구현 단계)
현 구조: Phaser 엔진 `game.js`(ExploreScene = world↔place), 데이터 `data/places.js`(`world{x,y}`,`theme`,`dialogue`,`lookMission`,`choices`,`piece`), UI `ui.js`(대사→사진→퀘스트→결과), `audio.js`, 표지 `index.html`+`style.css`.

**단계(파일 단위):**
1. **데이터 재구성** — `data/places.js`
   - 장소를 하루 동선 순서로 정리, `piece`→`card`(이름·의미) 필드화, `dayPhase` 태그 추가, `lookMission`을 일상 관찰형으로 교체, "학교 가는 길" 미션 장소 추가, 해수욕장=주말 해금 플래그.
2. **하루 진행 상태기계** — `game.js`(ExploreScene) + `ui.js`
   - `dayPhase` 상태, 단계 클리어 시 다음 구역 안개 해제·카드 적립. 월드 포탈의 `locked` 게이팅(이전 단계 미완료 시 잠금 + "아직 학교부터 가볼까?" 안내).
3. **안개(fog-of-war)** — `game.js`
   - 월드맵에 미발견 구역을 덮는 반투명 레이어(타일/Graphics), 방문 시 해당 권역 reveal(탐험감).
4. **결과 "하루지도"** — `ui.js`(`showResult`) + `index.html`/`style.css`
   - 보물지도 결과를 카드 기반 **내러티브 하루지도**로 교체(동적 문장).
5. **맵 품질 전환(도구)** — §8 (Tiled JSON) 또는 LPC 건물 아틀라스 적용.
6. **사진/카피** — `data/places.js` 대사·`photoCredit`·`lookMission`을 일상컷 기준으로.

### 8.2 핵심 판단
- 매니페스트는 "LPC(자연 32px)+Kenney(도시 16px) 혼합"을 제안하나, **32px↔16px 픽셀 밀도 불일치로 한 화면에 섞으면 어색**하다 → 혼합 비권장.
- "정착·일상 동네" 컨셉에는 **Kenney RPG Urban이 최적**: 횡단보도·버스정류장·상가·우체통·가로등·벤치 + **거리 이벤트용 자동차/버스** + **동네 주민 4방향 캐릭터**(아빠·친구·사서·시장 아주머니·행인 NPC 모두 충당) + **CC0**.
- 16px 우려는 카메라 줌(×3=48px)·픽셀아트 렌더로 해소. 풍부한 도시 타일이라 Tiny Town보다 가시성↑.

### 8.3 ✅ 확정 에셋 방향: **Kenney RPG Urban 단일 일관 세트** (사용자 승인 2026-06-24)
- 지형(잔디/물)·도로·횡단보도·건물·소품·자동차·캐릭터를 **한 스타일 16px**로 통일.
- 캐릭터도 Kenney 주민 스프라이트 사용(플레이어=아이, NPC=주민) → 스타일 일관 + CC0.
- LPC(external_v2)는 보류(중세풍·표기필수). 32px가 꼭 필요하면 추후 별도 결정.

### 8.4 맵 제작 방식
- 1차(빠름): `urban_tilemap_packed.png`(27열,16px)를 Phaser `spritesheet`로 로드 → `urban_tiles/`와 미리보기로 **용도별 타일 인덱스 표**(grass/road/crosswalk/roof/door/lamp…)를 만들어 `data/`의 맵 데이터에서 참조. 건물은 **멀티타일 조립**(지붕+벽+문) 또는 통짜 배치.
- 2차(품질): **Tiled(mapeditor.org)** 로 시각 저작 → Phaser `tilemapTiledJSON` 로드. 건물·포탈·NPC를 오브젝트 레이어로. (MCP 표준 없음 → Tiled가 그 역할)
- 제작 주체: 전담 맵-제작 에이전트가 인덱스 표 + 맵 데이터 저작(내 손코딩 약점 보완).

## 8.6 맵 현실성(realism) 방향 — 미결정
피드백: 절차적 타일 월드가 "너무 디자인적/추상적"이고 현실성이 없다. OpenAI 에셋 생성 가능(.env에 키 보유).
세 갈래(택1):

### A. OpenAI 생성 일러스트 동네 지도(배경) + 캐릭터·마커 오버레이 (가장 큰 변화, 현실성↑)
- gpt-image-1로 **속초 동네를 위에서 본 따뜻한 일러스트 지도 1장** 생성 → 월드맵 배경 이미지.
- 그 위에 캐릭터가 걷고(보이지 않는 **walkable/충돌 마스크** 별도 저작), 장소 마커 배치.
- 장: 즉시 "진짜 동네 지도" 느낌, 픽셀 추상감 해소. 단: 충돌 마스크·마커 정렬 수작업, 지리 정확도는 일러스트 수준.

### B. 생성 건물·사진 카드로 부분 현실성 (저위험, 엔진 유지)
- 타일 월드는 유지하되, 장소 건물 마커와 "진짜 모습 보기"에 **OpenAI 생성 실사풍 건물/거리 이미지**를 사용.
- 장: 보행/충돌 그대로, 현실성은 사람이 보는 핵심(건물·사진)에 집중. 단: 월드 바닥은 여전히 타일.

### C. 밀도 높은 실사풍 타일 마을 (Kenney Urban 최대 활용)
- 진행 중 Kenney Urban 타일로 도로·인도·블록·다양한 건물을 빽빽이 배치해 실제 도시 블록처럼.
- 장: 네이티브 보행, CC0. 단: 여전히 픽셀 게임 룩(“디자인적”이라는 지적이 남을 수 있음).

**권장:** 현실성을 가장 직접적으로 올리는 **A**(또는 A+B 결합). 단 A는 충돌 마스크 저작 비용 있음 → 하이브리드: **A로 월드 배경 + B로 건물/사진 + 상세맵은 타일** 도 가능.

### ✅ 확정: A+B 하이브리드 (사용자 승인 2026-06-24)
- 월드맵 = **OpenAI 생성 일러스트 동네 지도 배경** + 캐릭터 자유 보행(충돌은 맵 경계만, 물/건물은 트리거존으로) + 장소 트리거.
- 건물 마커·"진짜 모습 보기" = **OpenAI 생성 실사/일러스트 이미지**.
- 장소 상세맵 = 타일 유지(Kenney Urban).
- 구현: (1) `generate-assets.mjs`로 월드맵 1장 + 7개 일상컷 사진 생성·저장, (2) `game.js` 월드 씬을 타일맵→이미지 배경+자유보행+트리거존으로 교체, 좌표는 생성된 지도에서 눈으로 확정.

## 9. 수용 기준 (Acceptance Criteria, 테스트 가능)
- [ ] 월드맵에 6개 일상 구역이 권역으로 배치되고 핵심 동선이 길로 연결된다.
- [ ] 시작 시 미발견 구역은 안개로 가려지고, 방문하면 해당 구역 안개가 사라진다.
- [ ] `dayPhase`가 morning→…→weekend로 진행되며, 이전 단계 미완료 시 다음 핵심 포탈이 잠기고 안내 문구가 뜬다.
- [ ] 각 장소 완료 시 **일상 카드**가 적립되고 HUD에 표시된다.
- [ ] 결과 화면이 보물지도가 아니라 **카드 기반 하루지도 내러티브 문장**을 동적으로 출력한다.
- [ ] "학교 가는 길" 미션이 존재하고 횡단보도/버스정류장 관찰 항목을 포함한다.
- [ ] 해수욕장은 평일엔 잠겨 있고 하루 완료(밤) 후 **주말 탐험지로 해금**된다.
- [ ] NPC 중 판타지는 갈매기 1종이며 나머지는 동네 사람 톤이다.
- [ ] 사진 미션 항목이 일상 관찰형(입구·벤치·교문 등)으로 구성된다.
- [ ] 전체 하루 루프를 플레이로 완주 가능(브라우저에서 JS 에러 0).
