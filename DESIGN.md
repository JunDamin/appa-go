# 아빠, Go! — 맵 & 콘텐츠 설계 (정착 컨셉)

## 핵심 컨셉
관광이 아니라 **"새 동네 정착"**. 전학·이사로 속초에 온 5~9세 아이가 동네의
**일상 시설**(학교·어린이집·도서관·마트·놀이터·시장)을 미리 경험하고 친근해진다.
메타 서사: **엄마가 준 "동네 익히기 심부름"** 을 하나씩 해내며 **동네 지도 조각**을 모은다.

> 핵심 문장: *새 동네가 낯설기 전에, 먼저 익숙한 우리 동네가 되게 합니다.*

## 지도 구조 — 실제 속초 지리 반영 (OSM/행정동 기준)
속초 8행정동 구조를 단순화해 월드맵에 반영:

```
   서(설악산/노학동)                              동(동해 바다)
 ┌──────────────────────────────────────────────────────────┐
 │  ⛰숲        🐟속초관광수산시장(닭강정)     동명항   🌊해변 │ ← 북
 │                      〰 영랑호(영랑동·공원)                 │
 │              〰〰〰 청 초 호 (석호) 〰〰〰   🚤아바이마을  │ ← 중앙
 │        🛝놀이터(호수공원)                                  │
 │   ── 조 양 동 신 시 가 지 (우리 동네) ──                   │ ← 남
 │   🏫학교   🏠우리집   🧸어린이집   📚도서관   🛒마트       │
 └──────────────────────────────────────────────────────────┘
```

- **서**: 설악산(노학동) → 숲/나무
- **동**: 동해 바다, 속초해수욕장 → 바다 타일 + 배
- **중앙**: 청초호(석호) / **북동**: 영랑호(영랑동 주거·공원)
- **북중앙**: 시내·속초관광수산시장(닭강정)·동명항(동명동)
- **남(조양동 신시가지)**: **정착 가족의 생활권** — 우리 집·학교·어린이집·도서관·마트, 청초호 남쪽
- 청초호 어귀: 아바이마을·갯배(분위기)

## JRPG식 2단 구조 (대항해시대 항해/항구 모드)
- **월드맵(동네 탐험 모드)**: 속초 우리 동네를 카메라 추적으로 걸어다님. 건물 입구로 들어가면 →
- **장소 상세맵(항구 모드)**: 해당 시설의 상세 맵. NPC와 대화 → 사진+찾기미션 → 퀘스트 → 동네 지도 조각 획득 → 동네로 복귀.

## 장소 & 퀘스트 (정착·일상)
| 장소 | NPC | 일상 경험 / 퀘스트 | 조각 |
|------|-----|--------------------|------|
| 우리 초등학교 | 담임 선생님 | 학교 가는 길·교문·운동장 익히기 | 🏫 |
| 햇살 어린이집 | 어린이집 선생님 | 친구·장난감·낮잠 공간 보기 | 🧸 |
| 속초시립도서관 | 도서관 선생님 | 어린이실에서 책 빌리기 | 📖 |
| 동네 마트 | 마트 아저씨 | **엄마 심부름**: 우유·과일 사오기 | 🛒 |
| 청초호 놀이터 | 새 친구 | 미끄럼틀·그네, 첫 친구 사귀기 | 🛝 |
| 속초관광수산시장 | 시장 아주머니 | 닭강정 골목, 주말 가족 나들이 | 🍗 |

조각 6개를 모으면 **"우리 동네 지도 완성"** → 새 동네에 대한 친근감/자신감.

## 5~9세 UX 원칙
큰 버튼 · 짧은 문장 · 캐릭터 중심 대화 · 목소리로 읽어주기(Web Speech) · 사진 관찰 미션.
혼자 읽기를 전제하지 않고 보호자와 함께 보고 듣게 설계. 학교·아이 실제 사진은 개인정보
보호를 위해 일러스트/분위기 컷 사용.

## 기술
- Phaser 3 + 카메라 줌 추적, 타일맵, 4방향 걷기 스프라이트, 씬 페이드 전환
- DOM 오버레이(대사/사진/퀘스트/결과) + Web Speech 음성 + CC0 BGM/SFX
- 아트: 32px 고해상 CC0/CC-BY 타일셋·캐릭터로 교체 진행 중 (assets/external_v2)
- 데이터 주도: `data/places.js` 의 `world{x,y}`·`theme` 로 맵 자동 구성
