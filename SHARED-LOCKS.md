# SHARED-LOCKS — 공유 파일 점유 장부

공유 파일(`index.html`, `style.css`, `README.md`, `DESIGN.md`, `.gitignore`, `CLAUDE.md`)을 작업 세션이 수정해야 할 때 사용한다.

## 규칙
1. 점유: 아래 표에 한 줄을 추가하고 **커밋**한다(커밋이 직렬화 → 먼저 커밋한 세션이 소유).
2. 작업: 점유 중인 파일만 단독으로 수정한다.
3. 해제: 작업·커밋이 끝나면 상태를 `released`로 바꾸거나 줄을 지우고 커밋한다.
4. 이미 `held` 상태인 파일은 건드리지 않는다 — 해제될 때까지 대기하거나 통합 세션에 요청한다.

## 현재 락 (held 만 유효)

| 파일 | 세션/브랜치 | 시작시각(절대) | 목적 | 상태 |
|---|---|---|---|---|
| _(예시)_ index.html | session/ui | 2026-06-24 14:00 | 말풍선 DOM 추가 | released |

> 사용 후 예시 줄은 지워도 된다.
